# 👗 CLOSET ― じぶんの服でコーデ

自分の持っている服を登録して、**組み合わせを何通りもシミュレーション**できるコーディネートアプリです。

- **クローゼット**: 服を登録。**写真**(カメラ／画像 ― サムネイル化して端末に保存)でも、**しゅるい＋色＋がら**を選んで作るプロシージャル描画でもOK。カテゴリー(アウター/トップス/ボトムス/ワンピース/シューズ/こもの)で整理
- **コーデ**: アウター・トップス／ワンピース・ボトムス・シューズ・こもののスロットを ◀▶ で切り替え。**🎲シャッフル**で一発コーデ、**🔒スロット固定**で「このパンツは固定して上だけ変える」も。ワンピースを選ぶとボトムスは自動で隠れる
- **おきにいり**: 気に入ったコーデを保存して見返す
- 服も保存コーデも**ブラウザに保存**。プロシージャルな服は色や柄を再着色して描くので、写真がなくてもすぐ試せます(初回はサンプルの服が入っています ― 自分の服に入れかえてください)

# 🦊 どうぶつがっこう ― さんすう・こくご × どうぶつ

ある子どもの「**さんすう・こくご(けいさん)がもっとわかるアプリ**」と「**どうぶつのたべものやとくちょうがわかるアプリ**」という2つの願いを、1つに融合したこども向け知育アプリです。

- **さんすう**: たしざん・ひきざん、どうぶつが登場する文章題。大きな3択ボタンで直感的に
- **こくご**: もじの かず・はじめの おと・はんたいことば・なかまさがし(絵つき)
- **ごほうび**: 正解するとどうぶつと仲良くなり、数問ごとに**新しいどうぶつが登場**。その子が知りたかった **たべもの・とくちょう** が どうぶつ図鑑に増えていく(全20種)
- **やさしさ設計**: ぜんぶ ひらがな、大きなボタン、紙吹雪とほめ言葉、**こわい不正解なし**(まちがえても「もういちど！」、2回で答えをそっと教える)。⭐と進み具合は保存されます

> 手描きの絵は描けず泣いてしまった子の、言葉だけの願いから起こしたアプリです。学ぶ楽しさと、どうぶつを知るワクワクが一緒になりますように。

# 🐣 めばえ ― いきもの工房

手描きのUIスケッチ(いきものの作成・登録・図鑑)から着想した、**いきもの創造＆コレクション**アプリです。

- **こうぼう(工房)**: からだ・め・くち・あたま飾り・て・あし・もよう・いろ の8スロットを ◀▶ で選び、あなただけのいきものをプロシージャル生成。🎲でランダム、なまえをつけて「うみだす」
- **ずかん(図鑑)**: 生み出したいきものが一覧に。タップで詳細(種族・特徴・うまれ日)。「にがす」で手放すことも
- **おうち**: 登録したいきものが草原のおうちで歩きまわり、ぴょこぴょこ暮らす。タップすると跳ねて名前をふきだす
- すべてのいきものは遺伝子(genome)から**同じ描画関数**で描かれ、組み合わせは無限。図鑑・おうち・工房・詳細のどこでも同じ姿で登場します
- 作成したいきものは**ブラウザに保存**され、次回も残ります

# ☯ 縁起 ― Form vs Emptiness(対戦ゲーム)

「すべては空(くう)。だが縁(えん＝つながり／縁起)があるから形がある」という色即是空の思想を、**対戦ボードゲーム**にしたものです。

## ♟ ルール

ヘックス盤上のすべては、本来 **空**。形は **源(みなもと)への縁** があるときだけ存在できます。

- **色派**(あなた or AI): 手番に節点を灯し(隣の形へ自動で縁が結ばれる)、**3つの縁座(ゴール)すべてに形を灯せば勝ち**
- **空派**: **縁を1本断つ**。源へのつながりを断たれた形は **連鎖して滅する(縁滅)**。色の完成を19局まで防げば勝ち(=無常が勝つ)
- **断たれた縁は数手ののち結び直される**。だから**縁を輪(ループ)に厚く結べば、一手では崩れない** ― 「深い縁起ほど、形は強い」という戦略性が生まれます

### モード

- **ひとり ― 試練(ソロキャンペーン)**: あなたは色。**序 → 破 → 急 → 転 → 妙 → 空** の6ステージを、しだいに強くなる空(1手先読みの思考＋縁座を狙い撃つ否定戦略)＆縮む制限局数で攻略します。突破すると次の試練が開き、進行はブラウザに保存されます
- **自由対局**: 「色」「空」「観戦(AI対AI)」を選んで対局(互角設定 ・ AI対AIで色52%/空48%)

盤上のバランスは、断縁を「永続」にすると縁座が復活不能=色が構造的に必敗になることを検証で突き止め、**縁の回復(ヒール)** で解決しています(だから難易度は"局数と空の賢さ"で調整、"断つ数"は増やしません)。

## 🛠 技術メモ

- グラフ連結性(源からの到達可能性)を毎手 BFS で判定し、**連鎖崩壊(縁滅)** を計算
- 断縁の**回復(ヒール)**により、永続劣化ではなく動的な綱引きに(これがないと空が必ず勝つ設計だったのを検証で発見・修正)
- 色AIは **2連結の塊**を育てて複数の源に接続、空AIは**最も崩落を招く縁**を探索。合成音のシンギングボウル(形の生成)と重い余韻(縁滅)

# 🌏 TERRA — 生きている大地

地形を隆起させ、雨を降らせ、木を植え、動物を放つ。あとは眺めるだけで生態系が勝手に動き出す「箱庭テラリウム」です。

## 🌱 世界のしくみ

- **草 → ウサギ・シカ → キツネ** の食物連鎖が本当に演算されています。獲物が減ればキツネは飢え、捕食者が消えれば草食動物が草原を食べ尽くす ― 個体数は現実の生態系のように波打ちます
- **四季と昼夜**が巡ります。春は桜が咲いて花びらが舞い、夏の夜は蛍が光り、秋は紅葉が散り、冬は雪が積もって動物たちは冬ごもり。雪解け水は春に湖へ流れ込みます
- **水は物理シミュレーション**。雨は谷に集まり、湖をつくり、土を潤し、日照りには蒸発します
- **山火事**: 落雷(自然発生もします)で森が燃え、風向きで延焼し、雨が消し止めます
- 絶滅しても大丈夫。環境が回復すれば、動物たちは**外の世界から移住**してきます

## 🎮 操作(TERRA)

| ツール | 効果 |
|---|---|
| 🏔隆起 / ⛏掘削 | ドラッグで地形を変える(山を作ると寒くなり雪が積もる) |
| 🌧雨 / 🌱草 / 🌳木 | 環境を育てる |
| 🐇🦌🦊🐦 | 動物を放つ |
| ⚡雷 / ☄️隕石 | 破壊と再生 |
| 筆 / 速度(1〜8×) / 停止 | ブラシサイズ・時間操作 |

サウンドはすべてWeb Audio合成: 風・雨・小鳥のさえずり(昼)・虫の声(夏の夜)・雷鳴。

# 🌊 ABYSS — 深海10,911mへの潜航

海面からチャレンジャー海淵の最深部まで、実際の深度に沿って潜っていく没入型の深海探査体験です。

## 🤿 体験の流れ

- **潜航艇「ABYSS-01」を操縦**し、マウスで探照灯を照らしながら深海へ
- 深度に応じて実在の生物が登場 ― マイワシの群れから、リュウグウノツカイ、チョウチンアンコウ、ダイオウイカ、そして水深8,000mのシンカイクサウオまで
- **クリックで撮影**すると図鑑に記録。全26種(生物22種+ランドマーク4つ)のコンプリートを目指せます
- 深度とともに光が消え、水温が下がり、水圧が増していく。1,000mを超えると完全な闇 ― 頼りは探照灯と、生物自身の**生物発光**だけ
- 水深2,500mの**熱水噴出孔**、3,800mの**沈没船**、5,200mの**鯨骨生物群集**などのランドマークが待っています
- 光を怖がって逃げる魚、逆に光に誘われて近づいてくるチョウチンアンコウ、照らすと青い警報を光らせるクロカムリクラゲなど、生物ごとの反応の違いも
- 発見記録と最深到達深度は**ブラウザに保存**され、次回に引き継がれます

## 🎮 操作(ABYSS)

| 操作 | 効果 |
|---|---|
| **マウス** | 探照灯の向きと機体の操作 |
| **ホイール / W・S** | 潜航・浮上の速度調整 |
| **クリック** | 撮影(図鑑に記録) |
| **Space** | その場で停止 |
| **H** | UI表示切替 |

## ⚙️ 技術的なポイント(ABYSS)

- 26種すべての生物を**プロシージャルなCanvas描画**で表現(画像ファイルゼロ)。図鑑のサムネイルも同じ描画関数から動的生成
- `destination-out` 合成による**探照灯のライティング**と、闇を貫く生物発光のグローパス
- セグメントチェーンによるリュウグウノツカイ・フクロウナギの**リボン状の泳ぎ**
- Web Audio APIで**完全合成した環境音** ― 深度で変化する水中ノイズ、深海での船体のきしみ、クジラの鳴き声、ソナーピング
- 深度連動の色調・マリンスノー・海溝の岩壁・ゾーン演出、実話ベースの深度マイルストーン表示

# 🌌 GALAXY FORGE — リアルタイム銀河シミュレーター

最大 **12,000個の恒星** がリアルタイムに重力演算され、クリック＆ドラッグでブラックホールを射出して、銀河を自在に崩壊・衝突・再構築できます。

## 🎮 操作方法

| 操作 | 効果 |
|---|---|
| **クリック＆ドラッグ** | ブラックホールを射出(引いた方向と距離が初速になります) |
| **Space** | 時間の一時停止 / 再開 |
| **C** | すべてのブラックホールを消去 |
| **H** | UIの表示 / 非表示(スクリーンショット用) |

## 🪐 プリセット

- **渦巻銀河** — 3本腕のスパイラル銀河。中心核の周りを恒星がケプラー軌道で公転
- **銀河衝突** — 2つの銀河が衝突コースへ。核同士は合体してより巨大なブラックホールに
- **リング星雲** — 恒星がリング状に公転。摂動を与えると美しく崩壊します
- **カオス** — 無重力空間に恒星を散布。ブラックホールを投げ込んで銀河を"鍛造"してください

## 🎛 コントロール

- **恒星の数** (500〜12,000)
- **重力の強さ** / **時間の流れ** / **光の残像**(軌跡の長さ)
- **宇宙のサウンド** — Web Audio APIで完全合成したアンビエントドローン。ブラックホールの生成・合体時には質量に応じた音程のピングが鳴ります

## ⚙️ 技術的なポイント

- **Canvas 2D + 加算合成** による恒星のグロー表現と残像トレイル
- **Structure of Arrays** (`Float32Array`) による粒子データ管理で毎フレーム最大12,000粒子を60FPSで更新
- 色相バケツごとの **バッチ描画** で `fillStyle` 切り替えを最小化
- ソフトニング付き重力 + **サブステップ積分** で高速時間でも軌道が安定
- ブラックホール同士の **合体** と、事象の地平面に落ちた恒星の **降着**(質量が成長)
- **Web Audio API** で外部音源なしのアンビエントサウンドを合成
- レスポンシブ対応(モバイルはタッチ操作 + パネル折りたたみ)
