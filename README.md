# 🌏 TERRA — 生きている大地

地形を隆起させ、雨を降らせ、木を植え、動物を放つ。あとは眺めるだけで生態系が勝手に動き出す「箱庭テラリウム」です。

## 🌱 世界のしくみ

- **草 → ウサギ・シカ → キツネ** の食物連鎖が本当に演算されています。獲物が減ればキツネは飢え、捕食者が消えれば草食動物が草原を食べ尽くす ― 個体数は現実の生態系のように波打ちます
- **四季と昼夜**が巡ります。春は桜が咲いて花びらが舞い、夏の夜は蛍が光り、秋は紅葉が散り、冬は雪が積もって動物たちは冬ごもり。雪解け水は春に湖へ流れ込みます
- **水は物理シミュレーション**。雨は谷に集まり、湖をつくり、土を潤し、日照りには蒸発します
- **山火事**: 落雷(自然発生もします)で森が燃え、風向きで延焼し、雨が消し止めます
- 絶滅しても大丈夫。環境が回復すれば、動物たちは**外の世界から移住**してきます

## 🎮 操作(TERRA)

| ツール | 効果 |
|---|---|
| 🏔隆起 / ⛏掘削 | ドラッグで地形を変える(山を作ると寒くなり雪が積もる) |
| 🌧雨 / 🌱草 / 🌳木 | 環境を育てる |
| 🐇🦌🦊🐦 | 動物を放つ |
| ⚡雷 / ☄️隕石 | 破壊と再生 |
| 筆 / 速度(1〜8×) / 停止 | ブラシサイズ・時間操作 |

サウンドはすべてWeb Audio合成: 風・雨・小鳥のさえずり(昼)・虫の声(夏の夜)・雷鳴。

# 🌊 ABYSS — 深海10,911mへの潜航

海面からチャレンジャー海淵の最深部まで、実際の深度に沿って潜っていく没入型の深海探査体験です。

## 🤿 体験の流れ

- **潜航艇「ABYSS-01」を操縦**し、マウスで探照灯を照らしながら深海へ
- 深度に応じて実在の生物が登場 ― マイワシの群れから、リュウグウノツカイ、チョウチンアンコウ、ダイオウイカ、そして水深8,000mのシンカイクサウオまで
- **クリックで撮影**すると図鑑に記録。全26種(生物22種+ランドマーク4つ)のコンプリートを目指せます
- 深度とともに光が消え、水温が下がり、水圧が増していく。1,000mを超えると完全な闇 ― 頼りは探照灯と、生物自身の**生物発光**だけ
- 水深2,500mの**熱水噴出孔**、3,800mの**沈没船**、5,200mの**鯨骨生物群集**などのランドマークが待っています
- 光を怖がって逃げる魚、逆に光に誘われて近づいてくるチョウチンアンコウ、照らすと青い警報を光らせるクロカムリクラゲなど、生物ごとの反応の違いも
- 発見記録と最深到達深度は**ブラウザに保存**され、次回に引き継がれます

## 🎮 操作(ABYSS)

| 操作 | 効果 |
|---|---|
| **マウス** | 探照灯の向きと機体の操作 |
| **ホイール / W・S** | 潜航・浮上の速度調整 |
| **クリック** | 撮影(図鑑に記録) |
| **Space** | その場で停止 |
| **H** | UI表示切替 |

## ⚙️ 技術的なポイント(ABYSS)

- 26種すべての生物を**プロシージャルなCanvas描画**で表現(画像ファイルゼロ)。図鑑のサムネイルも同じ描画関数から動的生成
- `destination-out` 合成による**探照灯のライティング**と、闇を貫く生物発光のグローパス
- セグメントチェーンによるリュウグウノツカイ・フクロウナギの**リボン状の泳ぎ**
- Web Audio APIで**完全合成した環境音** ― 深度で変化する水中ノイズ、深海での船体のきしみ、クジラの鳴き声、ソナーピング
- 深度連動の色調・マリンスノー・海溝の岩壁・ゾーン演出、実話ベースの深度マイルストーン表示

# 🌌 GALAXY FORGE — リアルタイム銀河シミュレーター

最大 **12,000個の恒星** がリアルタイムに重力演算され、クリック＆ドラッグでブラックホールを射出して、銀河を自在に崩壊・衝突・再構築できます。

## 🎮 操作方法

| 操作 | 効果 |
|---|---|
| **クリック＆ドラッグ** | ブラックホールを射出(引いた方向と距離が初速になります) |
| **Space** | 時間の一時停止 / 再開 |
| **C** | すべてのブラックホールを消去 |
| **H** | UIの表示 / 非表示(スクリーンショット用) |

## 🪐 プリセット

- **渦巻銀河** — 3本腕のスパイラル銀河。中心核の周りを恒星がケプラー軌道で公転
- **銀河衝突** — 2つの銀河が衝突コースへ。核同士は合体してより巨大なブラックホールに
- **リング星雲** — 恒星がリング状に公転。摂動を与えると美しく崩壊します
- **カオス** — 無重力空間に恒星を散布。ブラックホールを投げ込んで銀河を"鍛造"してください

## 🎛 コントロール

- **恒星の数** (500〜12,000)
- **重力の強さ** / **時間の流れ** / **光の残像**(軌跡の長さ)
- **宇宙のサウンド** — Web Audio APIで完全合成したアンビエントドローン。ブラックホールの生成・合体時には質量に応じた音程のピングが鳴ります

## ⚙️ 技術的なポイント

- **Canvas 2D + 加算合成** による恒星のグロー表現と残像トレイル
- **Structure of Arrays** (`Float32Array`) による粒子データ管理で毎フレーム最大12,000粒子を60FPSで更新
- 色相バケツごとの **バッチ描画** で `fillStyle` 切り替えを最小化
- ソフトニング付き重力 + **サブステップ積分** で高速時間でも軌道が安定
- ブラックホール同士の **合体** と、事象の地平面に落ちた恒星の **降着**(質量が成長)
- **Web Audio API** で外部音源なしのアンビエントサウンドを合成
- レスポンシブ対応(モバイルはタッチ操作 + パネル折りたたみ)

## ✨ 遊び方(共通)

HTMLファイルをブラウザで開くだけです。

```bash
open abyss.html        # macOS
start abyss.html       # Windows

# またはお好みのサーバーで
npx serve .
```

(Settings → Pages → Branch: main / root)。
